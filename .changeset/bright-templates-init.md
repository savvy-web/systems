## Features

Initial release of @savvy-web/templates — pure TypeScript library for Silk Suite project scaffolding. Includes 10 template functions for generating package.json, tsconfig, biome, turbo, pnpm workspace, gitignore, changeset config, VS Code settings, README, and full workspace compositions. All templates use Effect Schema validation with no runtime I/O.
